# 🐍 Snake Rescue

A physics-based puzzle game where players create chain reactions to save the princess!

## Features

- 🎮 20 MVP Levels
- ⭐ Star Rating System
- 🔥 Physics Chain Reactions
- 📱 Mobile Optimized
- 🚀 Auto-Build via GitHub Actions

## Build

APK builds automatically on every push to `main` branch.
